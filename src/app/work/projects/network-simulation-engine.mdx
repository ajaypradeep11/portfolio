## Overview

This project was developed as part of a **Final Project in Networking**, where the classic **Tic Tac Toe** game is implemented with interactive controls and 3D-like gameplay.  

The game allows players to compete by placing **X** and **O** using custom mechanics (firing symbols instead of traditional clicks), while also supporting constraints and input handling for more dynamic interaction.  

The entire project was compiled and executed in **Release X64 mode** for performance and stability.

## Gameplay

- The objective is the same as classic **Tic Tac Toe**: align three **X** or **O** in a row to win.  
- Instead of simple clicks, players use a **weapon-like shooting system** to place their symbols.  
- The game features **pan controls**, constraints, and a more interactive board environment.
